TopologicalLayout.bendAdjacentLinks Field
See Also
 





Indicates whether to bend links that connect adjacent nodes or draw them as straight lines.

Namespace: MindFusion.Graphs
File: TopologicalLayout.js

 Syntax

JavaScript  Copy Code

var bendAdjacentLinks

 Remarks

true to bend links connecting adjacent nodes, or false to draw them as straight lines. The default value is false.

This property applies only to forward links. Backward links between adjacent nodes are always rendered bent.
