Licensing

MindFusion.Diagramming licenses are perpetual and royalty-free. There are no separate trial / licensed builds of MindFusion scripts provided, instead you disable trial mode by specifying a license key string. When you purchase the component, you can find its license key string listed on the Keys & Downloads page at the customer's portal. Copy the key to a diagram_lic.txt file in same folder where the diagram html page is located. The file name and location can be changed by calling the diagram's setLicenseLocation method. Alternatively, the key string can be set directly by calling setLicenseKey method.
